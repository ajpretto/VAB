# Capabilities Research — 2026-03-05

## Top Trending AI Projects (March 2026)

### 1. OpenClaw (formerly Clawdbot) — 250k+ stars
- **Status:** Just surpassed React as most starred JS project
- **Latest:** Major platform update shipped March 4, 2026
- **ClawHub:** 5,700+ community-built skills
- **Security note:** Cisco found vulnerabilities - audit skills before using

### 2. Ollama — 162k stars
- Local LLM runner, one-command setup
- New `ollama launch` integrates with Claude Code, Codex, OpenCode
- **For Anthony:** Run local models for testing without API costs

### 3. DeepSeek-R1 — 80k+ stars
- Open-source reasoning model rivaling OpenAI o1
- ~$0.55/million tokens (90% cheaper than competitors)
- Sizes from 1.5B to 70B parameters
- **For Anthony:** Could reduce AI costs for voice assistant backend

### 4. n8n — 174k stars
- Workflow automation with AI agent nodes
- 400+ app integrations
- Self-hosted = unlimited free executions
- **For Anthony:** Could automate lead tracking, follow-ups, CRM updates

### 5. Dify — 130k stars
- Visual drag-and-drop AI workflow builder
- Built-in RAG, supports 100+ LLM providers
- **For Anthony:** Could build customer onboarding flows visually

### 6. Browser Use — 76k stars
- Give any LLM browser control via Playwright
- Navigate, click, fill forms, extract data
- **For Anthony:** Could automate lead research, find owner info

### 7. Mastra — 21k stars
- TypeScript-native agent framework
- "Observational Memory" cuts token costs 4-10x
- From Gatsby.js team
- **For Anthony:** If building custom agents, consider this for cost savings

### 8. GLM-5 — New
- 744B parameter MoE model
- Lowest hallucination rate of any model tested
- SWE-bench 77.8, approaching Claude Opus
- MIT license

### 9. WebMCP — Emerging W3C Standard
- Websites expose callable tools to AI agents
- Chrome 146 has early implementation
- **For Anthony:** Future-proof - AI agents will interact with websites natively
