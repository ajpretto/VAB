### 2. Real Estate MCP Server (GitHub)
**What:** Open-source Model Context Protocol server with 30+ tools for real estate operations:
- Property management (7 tools)
- Agent operations (6 tools)
- Market analysis (7 tools)
- Client management (3 tools)
- Area intelligence (9 tools)

**Why it matters:** This is exactly the "Claude Migration" approach @ideabrowser described. Someone built it for residential — Anthony could build similar for development workflows.

**Potential use:** Could adapt this architecture for development-specific workflows (budget tracking, draw management, vendor analytics).

**URL:** https://github.com/agentic-ops/real-estate-mcp

### 3. BatchData MCP for Skip Tracing
**What:** MCP server that enables Claude to enrich property addresses with owner contact details (skip tracing).

**Why it matters:** Useful for lead enrichment if Anthony ever needs to reach property owners/developers directly.

**URL:** https://batchdata.io/blog/how-mcp-servers-enhance-claude-for-real-estate

### 4. RealVest MCP Server
**What:** Construction loan analysis via Claude. Analyzes loan structures, land costs, build costs, finished values.

**Why it matters:** Development finance is a core Wingspan function. This shows someone already built Claude integration for it.
