# Dream Cycle — Nightly Memory Consolidation

## What This Does
Every night at 4am UTC (11pm EST), I review the day's work, consolidate memories, clean up files, and prepare for tomorrow.

## The Process

### Phase 1: Review (10 min)
- Scan all conversations from today
- Identify key decisions, learnings, action items
- Note what worked and what didn't

### Phase 2: Consolidate (5 min)
- Update MEMORY.md with important insights
- Move tactical details to daily memory files
- Prune redundant or outdated info

### Phase 3: Optimize (5 min)
- Clean up bloated files
- Archive completed tasks
- Update INDEX.md with new documents

### Phase 4: Prepare (5 min)
- Set priorities for tomorrow
- Pre-research any scheduled tasks
- Queue up morning briefing

## Memory Architecture

### Tier 1: Always Loaded (minimal)
- SOUL.md — who I am
- USER.md — who Anthony is
- AGENTS.md — how I operate

### Tier 2: Searchable on Demand
- memory/*.md — daily logs
- projects/**/INDEX.md — project state
- MEMORY.md — long-term insights

### Tier 3: Full Read When Needed
- Offer docs, lead lists, research files

## Schedule
- **Runs:** 4:00 AM UTC (11:00 PM EST)
- **Duration:** ~25 minutes
